# IISER Bhopal Sports Council Website

A modern, responsive website for the IISER Bhopal Sports Council featuring dynamic content, beautiful UI, and comprehensive sports information.

## Features

- **Dynamic Hero Slider**: Auto-sliding hero images with pause-on-hover and manual navigation
- **Achievement Ribbon**: Scrolling showcase of sports achievements
- **News & Announcements**: Featured news with dedicated news page
- **Upcoming Events**: Detailed event listings with registration options
- **Photo Gallery**: Masonry-style gallery with hover effects
- **Responsive Design**: Fully responsive across all devices
- **Modern UI**: Glass-morphism effects, smooth animations, and gradient accents

## Pages

- `index.html` - Homepage with hero, news preview, and events
- `news.html` - Full news listing with filters
- `events.html` - Detailed events page with registration
- `gallery.html` - Photo gallery with masonry layout

## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- Modern design patterns (Glass-morphism, Gradients)

## Setup

1. Clone the repository
2. Open `index.html` in a web browser or serve with a local server:
   ```bash
   python3 -m http.server 8080
   ```
3. Navigate to `http://localhost:8080`
